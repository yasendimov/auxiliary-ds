## Design Systems

- [Carbon Design System (IBM)](https://carbondesignsystem.com/) — closest visual relative, Swiss-modernist heritage, Productive/Expressive duality
- [Radix Themes](https://www.radix-ui.com/) — component API vocabulary, variant weight ladder, color prop patterns
- [OpenBridge](https://www.openbridge.no/) — maritime/industrial operational interface patterns, safety-first HMI
- [Vercel Geist](https://vercel.com/geist/introduction) — application-tier UX architecture, component consistency, color hierarchy
- [DevRev Standard](https://devrev.standard-projects.com/) — modern design system approach

- [Area 17](https://area17.com/) — numeric precision, pixel-level rendering discipline, mathematical systems
- [Area 17 Design Techniques](https://guides.area17.com/design-techniques) — practical design methodology
- [Area 17 Brand 2024](https://guides.area17.com/a17-brand-2024#logo) — brand identity craft
- [Area 17 Foundations of Feedback](https://guides.area17.com/foundations-of-feedback) — feedback methodology
- [Rauno Freiberg](https://rauno.me/) — interface craft, attention to detail
- [Devouring Details](https://devouringdetails.com/) — micro-interaction and polish reference
- Swiss graphic design masters — Müller-Brockmann, Hofmann, Ruder
- Karl Gerstner's layout grid — programmatic design methodology
- US Unigrid system — Massimo Vignelli's systematic grid for the National Park Service
